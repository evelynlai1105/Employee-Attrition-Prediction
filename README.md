# Employee Attrition Prediction

利用機器學習分析員工離職因素，建立離職預測模型，協助企業提早辨識高風險員工，作為人力資源決策參考。

---

## Project Overview

員工流動率會增加企業招募、教育訓練及管理成本，也可能影響團隊績效與組織穩定。

本專案透過資料分析與機器學習方法，分析影響員工離職的關鍵因素，建立預測模型，提供企業更具依據的人才留任策略。

---

## Business Problem

本專案希望協助回答以下問題：

- 哪些員工具有較高離職風險？
- 哪些因素最容易影響離職？
- 如何運用資料分析協助 HR 制定留才策略？
- 如何建立資料驅動的人才管理模式？

---

## Dataset

分析資料包含：

- 員工基本資料
- 部門資訊
- 職務(Job Role)
- 工作年資
- 月薪(Monthly Income)
- 加班狀況(OverTime)
- 工作滿意度(Job Satisfaction)
- 工作生活平衡(Work-Life Balance)
- 績效評估
- 環境滿意度
- 其他人力資源相關欄位
